## 🛠️ How to Run the Scripts Locally

To run any of the Python scripts on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HP04Harsh/python-dsa-crusade.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd python-dsa-crusade
   ```
3. **Run a script using Python 3:**
   ```bash
   python script_name.py
   ```
